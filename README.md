## heroku-server

Google Homeを買い物に役立てる用に作成したAPIサーバ。

https://speakerdeck.com/fuwamaki/googlehomewomai-iwu-niyi-li-teru

### 開発環境

- node v9.3.0
- express v4.15.5
- mongodb v3.0.1

### heroku

heroku環境設定方法。
herokuにログインします。

```
$ heroku login
```

herokuにデプロイ。

```
$ git push heroku [branchName]
```
